## Add Dependency
Download the latest AAR from [Maven Central][2] or grab via Gradle:
```groovy
implementation 'com.luckylittlesparrow:waterwall:1.0.0'
```
or Maven:
```xml
<dependency>
  <groupId>com.github.luckylittlesparrow</groupId>
  <artifactId>waterwall</artifactId>
  <version>1.0</version>
  <type>pom</type>
</dependency>
```
